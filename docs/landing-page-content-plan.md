#### 交互设计
- **触发方式**: 点击页面任意位置
- **触发概率**: 20%（每次点击有 20% 的概率触发）
- **显示位置**: 鼠标点击位置附近
- **动画效果**: 从点击位置弹出，淡入淡出
- **显示时长**: 2 秒后自动消失
- **样式设计**: 气泡样式，带有动画效果

#### 技术实现
```javascript
// Slogan 数组
const slogans = [
  "你今天动了吗？",
  "活着，就是为了做动画",
  "别让你的动画'死'在半路上",
  // ... 更多 slogan
];

// 点击事件
document.addEventListener('click', (e) => {
  // 20% 概率触发
  if (Math.random() < 0.2) {
    const randomSlogan = slogans[Math.floor(Math.random() * slogans.length)];
    showSlogan(e.clientX, e.clientY, randomSlogan);
  }
});

// 显示 slogan
function showSlogan(x, y, text) {
  const slogan = document.createElement('div');
  slogan.className = 'slogan-popup';
  slogan.textContent = text;
  slogan.style.left = x + 'px';
  slogan.style.top = y + 'px';
  document.body.appendChild(slogan);
  
  // 动画效果
  gsap.fromTo(slogan, 
    { opacity: 0, scale: 0.5, y: 20 },
    { opacity: 1, scale: 1, y: 0, duration: 0.3, ease: 'back.out' }
  );
  
  // 2秒后消失
  setTimeout(() => {
    gsap.to(slogan, {
      opacity: 0,
      scale: 0.8,
      duration: 0.3,
      onComplete: () => slogan.remove()
    });
  }, 2000);
}
```

#### 样式设计
```css
.slogan-popup {
  position: fixed;
  background: linear-gradient(135deg, #ff6b35, #ff8c42);
  color: white;
  padding: 12px 24px;
  border-radius: 20px;
  font-size: 16px;
  font-weight: 600;
  pointer-events: none;
  z-index: 9999;
  box-shadow: 0 4px 15px rgba(255, 107, 53, 0.4);
  white-space: nowrap;
}
```

#### 注意事项
- Slogan 不会在 CTA 按钮点击时触发
- Slogan 不会在 FAQ 点击时触发
- Slogan 不会在导航链接点击时触发
- 移动端可以适当降低触发概率（10%）
